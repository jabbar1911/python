### Prerequisites

- **Python 3.8+** installed on your system
- Basic command-line knowledge
- A code editor (VS Code, PyCharm, or any IDE)

### Clone the Repository

```bash
git clone https://github.com/jabbar1911/python.git
cd python
```

### Running Daily Practice Scripts

```bash
# Navigate to any day's folder and run
cd python-practice/Day_1
python Day_1.py

# Example: Run Day 35 (Advanced List Operations)
cd python-practice/Day_35
python Day_35.py
```

```bash

# Example: Run the Password Generator
cd python-mini-projects/03_password_generator
```
